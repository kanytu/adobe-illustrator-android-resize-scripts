adobe-illustrator-android-resize-scripts
========================================

Some scripts for Android resource resizing


`Export Android PNGs.js` - This script will the export the resized PNG's for the current document. You will be prompted a folder and the baseline (mdpi) width and heigh. No changes will be done in the document.


`Export Android PNGs - ALL.js` - Works the same way as `Export Android PNGs` but for all the opened documents.
